== Installation ==

= From within dashboard (recommended) =

1. Navigate to `Dashboard > Plugins > Add New`;
2. Search for `Simple Lead Generator`;
3. Click `Install`, then `Activate`.

= Manual installation =

1. Download the plugin as a .zip file;
2. Unzip downloaded archive and upload `simple-lead-generator` folder under your `/wp-content/plugins/` directory (resulted plugin path should be `/wp-content/plugins/simple-lead-generator/**`);
3. Navigate to `Dashboard > Plugins` and activate the plugin.
